<decisions_made>

- **ErrorBoundary wrapper pattern**: clase inner `ErrorBoundaryInner` con prop `t: (key) => string`; export default es functional wrapper que inyecta `useTranslation()`. Necesario porque React ErrorBoundary requiere clase pero hooks solo funcionan en componentes funcionales.
- **Tailwind tokens eliminados en lugar de migrar**: `surface`/`night`/`brand` duplicaban zinc/amber estándar. Migrar el código habría sido refactor de 200+ líneas sin beneficio visual.
- **ArgoCD sync via kubectl rollout restart**: `argocd` CLI no tenía credenciales disponibles en shell. `values-prod.yaml` ya tenía `tag: v0.3.12` (el CD lo actualizó); el rollout restart fue suficiente para que ArgoCD recogiera el nuevo estado.
- **`bg-amber-600` con `!important`**: necesario porque Tailwind genera `.bg-amber-600` con especificidad de clase (0,1,0); nuestro `[data-theme="light"] .bg-amber-600` tiene (0,1,1) pero el `!important` garantiza que no haya sorpresas con utility classes combinadas.
</decisions_made>
